DbFit

Test-driven database development. Write readable, easy-to-maintain unit and integration tests for your database code.

The complete solution

Write, manage and execute tests directly from the browser.

Free and open source

Released under the GPL license, source available on GitHub.

Built with FitNesse

DbFit is written on top of FitNesse, a mature, fully-featured framework with a large community.

Supports all major DBs

Out-of-the-box support for Oracle, SQL Server, MySQL, DB2, PostgreSQL, HSQLDB and Derby.

Readable tests

DbFit tests are written using tables, making them more readable than xUnit-style tests.

Automatable

Run tests from the commandline, any Java IDE or CI build tool.
